# Kitty Config

To install, clone this repo into `~/.config/kitty`.

The custom icon [should be automatically installed](https://sw.kovidgoyal.net/kitty/faq/#i-do-not-like-the-kitty-icon), though the MacOS Dock cache might need to be cleared:

```bash
rm /var/folders/*/*/*/com.apple.dock.iconcache; killall Dock
```
